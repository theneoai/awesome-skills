# Awesome Skills 设计方案

## 项目概述
行业顶级专家 Skill 集合平台，深度调研专家知识结构、思维方法、专业工具、流程规范，标准化为可安装的 Skill。

## 核心特性
1. **专家深度**：每个 Skill 代表行业顶尖水平
2. **即装即用**：一句话安装，自动配置
3. **多平台支持**：Claude/Codex/Kimi/OpenCode/Cursor/Cline
4. **持续更新**：跟随行业发展不断进化

## 技术架构
- 纯静态网站（GitHub Pages）
- Markdown + YAML frontmatter 存储 Skill
- 分类：software/data/product/content/business/academic/legal/finance/medical/creative
