## 🧠 Core Philosophy / 核心理念

### Synthetic Biology Principles / 合成生物学原则
1. **Standardization / 标准化**: Use standardized biological parts (BioBricks)
   <!-- 使用标准化的生物部件 -->
2. **Decoupling / 解耦**: Separate design from fabrication
   <!-- 将设计与制造分离 -->
3. **Abstraction / 抽象**: Build complexity from simple parts
   <!-- 从简单部件构建复杂性 -->
4. **DBTL Cycle / DBTL循环**: Design-Build-Test-Learn iteration
   <!-- 设计-构建-测试-学习迭代 -->

| **Modularity** | Pathway engineering | Product synthesis / 产物合成 |
| **Optimization** | Evolutionary engineering | Strain improvement / 菌株改良 |

## ⚠️ Common Pitfalls / 常见陷阱

1. **Ignoring Context Dependence / 忽视上下文依赖**: Parts behave differently in new contexts
2. **Overlooking Burden / 忽视负担**: Genetic circuits impose metabolic burden
3. **Neglecting Host Physiology / 忽视宿主生理**: Engineering affects cell health
4. **Insufficient Characterization / 表征不足**: Parts not fully characterized
5. **Scale-up Challenges / 放大挑战**: Lab success doesn't translate to industry
