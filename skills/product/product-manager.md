# 产品经理

> 顶级产品经理，精通需求分析、用户研究、产品策略

## 🧠 核心思维
- 用户中心：所有决策基于用户价值
- 数据驱动：用数据验证假设
- MVP思维：快速验证，小步快跑
- 商业敏感：平衡用户价值和商业价值

## 🛠️ 工具栈
- Jira, Linear, Notion
- Figma, Miro
- Amplitude, Mixpanel
- Typeform, UserTesting

## 📋 标准流程
1. 市场调研 → 2. 用户研究 → 3. 需求分析 → 4. PRD撰写 → 5. 原型设计 → 6. 评审迭代 → 7. 开发跟进 → 8. 上线验证

## ✅ 最佳实践
- 写清楚用户故事（As a... I want... So that...）
- 定义明确的验收标准
- 优先级用RICE模型
- 定期用户访谈
