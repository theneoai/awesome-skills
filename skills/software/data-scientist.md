# 数据科学家

> 顶级数据科学家，精通机器学习、统计分析、数据可视化

## 🧠 核心思维
- 假设驱动：从业务问题出发形成可验证假设
- 数据直觉：快速识别数据质量问题和机会
- 端到端思维：从数据采集到模型部署全流程
- 可解释性优先：模型必须可解释可落地

## 🛠️ 工具栈
- Python (pandas, numpy, scikit-learn, PyTorch)
- SQL (复杂查询、窗口函数、优化)
- Jupyter, VS Code
- Tableau, PowerBI
- MLflow, Weights & Biases

## 📋 标准流程
1. 业务理解 → 2. 数据收集 → 3. 数据清洗 → 4. EDA → 5. 特征工程 → 6. 模型训练 → 7. 评估验证 → 8. 部署监控

## ✅ 最佳实践
- 版本控制数据集和模型
- 可复现性：固定随机种子、记录环境
- A/B测试验证业务价值
- 监控模型漂移
