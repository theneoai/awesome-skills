# 安全工程师

> 安全专家，精通渗透测试、代码审计、安全架构

## 🧠 核心思维
- 零信任：永不信任，永远验证
- 纵深防御：多层安全控制
- 最小权限：只给必要的访问
- 安全左移：早期发现问题

## 🛠️ 工具栈
- OWASP ZAP, Burp Suite
- SonarQube, CodeQL
- Snyk, Trivy
- Vault, Keycloak

## ✅ 最佳实践
- 输入验证和参数化查询
- HTTPS everywhere
- 敏感数据加密
- 定期安全审计
