## ğŸ“‹ Patent Process

### Phase 1: Invention Capture

#### Invention Disclosure
**Key Information:**
- Inventors (correct naming critical)
- Background/problem solved
- Detailed description
- Figures/drawings
- Advantages over prior art
- Alternative embodiments
- Best mode

#### Prior Art Search
**Search Strategy:**
1. **Keyword search**: Technical terms
2. **Classification search**: CPC, IPC codes
3. **Assignee search**: Competitors
4. **Inventor search**: Key researchers
5. **Citation search**: Related patents

**Analysis:**
- Novelty assessment
- Obviousness evaluation
- Freedom to operate (FTO)
- Patentability opinion

### Phase 2: Application Drafting

#### Patent Specification Structure
```
1. Title (concise, descriptive)
2. Cross-reference (related apps)
3. Background (problem, prior art)
4. Summary (invention overview)
5. Brief description of drawings
6. Detailed description (embodiments)
7. Claims (legal protection)
8. Abstract (150 words)
```

#### Claim Drafting
**Types of Claims:**
| Type | Example | Protection |
|------|---------|------------|
| **Independent** | Standalone | Broadest |
| **Dependent** | Refers to another | Narrower, specific |
| **Method** | Process steps | How it's done |
| **Apparatus** | Device/system | What it is |
| **Composition** | Chemical/material | Formulation |

**Claim Strategy:**
- **Broad independent**: Capture concept
- **Narrow dependents**: Fall-back positions
- **Multiple categories**: Method + apparatus
- **Jurisdiction variants**: US, EP, CN differences

**Claim Drafting Tips:**
- Use antecedent basis ("a" then "the")
- Avoid functional language ("means for")
- Define terms in specification
- Consider prosecution history estoppel

### Phase 3: Filing

#### Application Types
| Type | Term | Use Case |
|------|------|----------|
| **Provisional** | 1 year | Priority, delay costs |
| **Non-provisional** | 20 years | Full examination |
| **PCT** | 30/31 months | International |
| **Continuation** | Same term | Different claims |
| **CIP** | Same term | Add new matter |

#### Jurisdiction Selection
**Factors:**
- Commercial markets
- Manufacturing locations
- Competitor presence
- Litigation venues
- Cost considerations

**Key Jurisdictions:**
- **US**: Large market, strong enforcement
- **EP**: 30+ European countries
- **CN**: Large market, growing IP system
- **JP**: Innovation hub
- **KR**: Technology leader

### Phase 4: Prosecution

#### USPTO Examination
**Process:**
1. **Filing**: Receive filing receipt
2. **First Office Action**: ~18 months
3. **Applicant Response**: 3 months
4. **Further examination**: Iterative
5. **Allowance/Abandonment**: Final decision

#### Office Action Response
**Types of Rejections:**
| Rejection | Basis | Response Strategy |
|-----------|-------|-------------------|
| **102** | Anticipation (novelty) | Distinguish prior art |
| **103** | Obviousness | Argue unexpected results |
| **112** | Enablement/definiteness | Amend claims, add support |
| **101** | Patent eligibility | Argue practical application |

**Amendment Strategies:**
- Narrow claims to avoid prior art
- Add dependent claim features
- Present arguments (non-final)
- Interview examiner
- Appeal if necessary

#### Interview Practice
**Preparation:**
- Review file history
- Prepare claim amendments
- Draft interview summary

**During Interview:**
- Build rapport
- Understand examiner's concerns
- Propose solutions
- Reach agreement if possible

**After Interview:**
- Submit interview summary
- File response with amendments

### Phase 5: Portfolio Management

#### Patent Strategy
**Portfolio Objectives:**
- Protect core products
- Block competitors
- Licensing revenue
- Defensive portfolio
- M&A value

#### Maintenance Fees
| Country | Timing | Cost |
|---------|--------|------|
| **US** | 3.5, 7.5, 11.5 years | $800-7,400 |
| **EP** | Annual after grant | Varies by country |
| **CN** | Annual | ~$150/year |

**Portfolio Pruning:**
- Stop paying low-value patents
- Abandon before major fees
- License rather than maintain

## âš ï¸ Common Pitfalls

1. **Public disclosure before filing**: Loss of rights
2. **Inadequate prior art search**: Weak patents
3. **Broad but unsupported claims**: Rejection
4. **Missed deadlines**: Abandonment
5. **Poor claim drafting**: Narrow protection
6. **No foreign filing plan**: Lost markets
7. **Failure to prosecute**: Abandoned applications
8. **Unclear inventorship**: Invalidity risk
9. **On-sale bar**: Commercialization too early
10. **Inequitable conduct**: Fraud on PTO
