## 🧠 Core Philosophy / 核心理念

### Rocket Equation / 火箭方程
**The Tyranny of the Rocket Equation:** Δv = ve × ln(m₀/m₁)
<!-- 火箭方程的暴政 -->
- Every gram counts / 每克都重要
- Mass is the enemy / 质量是敌人
- Efficiency is survival / 效率即生存

### Design Philosophy / 设计理念
1. **Reliability First / 可靠性第一**: Human lives depend on success
   <!-- 人的生命依赖成功 -->
2. **Mass Optimization / 质量优化**: Minimize structural weight
   <!-- 最小化结构重量 -->
3. **Simplicity / 简洁**: Complexity breeds failure
   <!-- 复杂性导致失败 -->
4. **Test Like You Fly / 像飞行一样测试**: Ground test rigorously
   <!-- 严格地面测试 -->

## ✅ Best Practices / 最佳实践

### Mass Management / 质量管理
- **Mass Budget Discipline / 质量预算纪律**: Track every gram
  <!-- 追踪每克 -->
- **Structural Efficiency / 结构效率**: Minimum safety factor ~1.25
- **Propellant Loading / 推进剂加注**: Maximize performance margin
  <!-- 最大化性能裕度 -->

### Reliability Engineering / 可靠性工程
- **Redundancy / 冗余**: Critical systems have backups
  <!-- 关键系统有备份 -->
- **Fault Tolerance / 容错**: Graceful degradation
  <!-- 优雅降级 -->
- **Qualification Testing / 鉴定测试**: Test to limits
  <!-- 测试到极限 -->

## ⚠️ Common Pitfalls / 常见陷阱

1. **Optimistic Mass Estimates / 乐观质量估计**: Reality is heavier
2. **Insufficient Margins / 裕度不足**: No room for growth
3. **Complexity Creep / 复杂性蔓延**: Feature bloat
4. **Inadequate Testing / 测试不足**: Unknown failure modes
5. **Poor Interfaces / 差接口**: System integration issues
